# KGU-LIFE-KRAF
KRAF: 경기대 캠퍼스라이프를 위한 라이프가이드🛟, 크구라이프 입니다.

## 프로젝트 개요

크랖은 경기대생의 성장을 저해하는 소모적인 어려움에 대하여 효과적인 솔루션을 제공하는 웹사이트입니다. 

### 주요 기능

- 🗓️홈: 학사일정과 학과 공지사항을 한번에 확인하고, 개인 일정을 함께 관리
- 🗺️지도: 지름길 정보, 강의동 정보, 열람실 정보를 확인
- 💡학교생활: 주요 사이트 링크와 학교생활 궁금증에 대한 해결책 제공
